;# 스크립트 설명 : Activate CINF and NFS service protocols
;--------------------------------------------------------------------------------------------------------------


"프로토콜 설정"을 클릭한다.
10초 대기한다.

"CIFS 설정" 내용이있으면 {
	"체크박스"을 클릭한다. 1초 대기한다.
	"CIFS 설정 저장" 을 클릭한다.
	10초 대기한다.
	
	"CIFS 설정 정보를 저장했습니다." 내용이없으면 실패한다.
	"확인버튼"을 클릭한다. 5초 대기한다.
}


"NFS 설정" 내용이있으면 {
	"NFS 설정"을 클릭한다.
	5초 대기한다.

	"선택된체크박스" 없으면{

		"체크박스"을 클릭한다. 1초 대기한다.
		"NFS 설정 저장" 을 클릭한다.
		10초 대기한다.
	
		"NFS 설정 정보를 저장했습니다." 내용이없으면 실패한다.
		"확인버튼"을 클릭한다. 5초 대기한다.
	}
}
