### ID = 볼륨 관리 001

  > 사전조건 - 볼륨 풀 동적할당이 있어야함

### ID = 볼륨 관리 004

 * 아비터 볼륨 생성 (볼륨 타입: 고정할당, 볼륨 명: vol4, 전송 유형: tcp, 분산 정책: Distributed, 복제 수: 2, 체인모드)

### ID = 볼륨 관리 007
