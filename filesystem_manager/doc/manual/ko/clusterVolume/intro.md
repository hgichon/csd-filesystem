# 2 클러스터 볼륨 관리

AnyStor-E는 스토리지 서비스의 성능 및 안정성을 최적으로 구성하기 위해 리눅스 LVM 볼륨 매니저와 **GlusterFS** 파일 시스템을 기본으로 탑재하였습니다.

볼륨 풀은 클러스터 노드의 로컬 LVM 영역을 클러스터에서 통합 관리하여 사용하며, 다양한 볼륨 풀 유형에 따른 클러스터 볼륨을 제공합니다.

**스냅샷을 비롯한 씬-프로비저닝 볼륨**을 구성할 수 있으며, **네트워크 RAID 볼륨**을 구성하여 백업 및 아카이빙 시 스토리지 활용율을 높일 수 있습니다.

| 구분         | 설명             |
| ------------ | ---------------- |
| **볼륨 풀 관리** | 클러스터의 각 노드별 LVM 물리 볼륨/볼륨 그룹/논리 볼륨을 통합 관리하는 기능으로 <br>**볼륨 관리** 기능을 사용하기 위해 미리 생성, 설정되어야 합니다. |
| **볼륨 관리** | 각 노드의 볼륨 풀에서 생성된 논리 볼륨을 사용한 클러스터 볼륨을 생성/삭제/확장할 수 있으며 <br>스냅샷 등의 부가적인 기능을 수행할 수 있습니다. |
| **스냅샷 스케줄링 관리** | 클러스터 볼륨의 스냅샷을 주기적으로 생성하는 스케줄링을 설정/삭제할 수 있습니다. |
